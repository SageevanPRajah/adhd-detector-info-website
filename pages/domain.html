Literature Survey

Background & Prior Work

ADHD (Attention Deficit Hyperactivity Disorder) is one of the most common neurodevelopmental disorders in children, affecting approximately 5-10% of the global child population. Current diagnostic procedures rely heavily on behavioral questionnaires and clinical observations, which are inherently subjective and resource-intensive.

📊

Traditional Methods

DSM-5 criteria-based assessments using Conners Rating Scales and clinical interviews, requiring 4-6 hours of specialist time per child.

🧠

Neuroimaging Approaches

fMRI and EEG-based studies have identified neural markers of ADHD, but high cost and accessibility limit clinical deployment.

💻

Computational Methods

Machine learning models trained on single behavioral modalities (eye movement alone, or handwriting alone) show promising results but lack integration.

🎮

Gamification in Diagnosis

Child-friendly gamified tasks increase data quality and reduce assessment anxiety, improving the reliability of behavioral measurements.

Research Gap

What's Missing

Existing ADHD detection methods rely on time-consuming manual clinical observations and rarely integrate multiple behavioral indicators into a single automated system.

  • No unified multimodal behavioral analysis platform exists
  • Individual modalities (speech, eye-tracking) studied in isolation
  • Limited child-appropriate interfaces for data collection
  • Lack of real-time AI prediction in clinical settings
  • High cost barriers prevent widespread screening

Research Problem

How can a multimodal AI system integrating eye-tracking, posture analysis, speech patterns, and handwriting behavior provide an accurate, accessible, and efficient early screening tool for ADHD in children, reducing dependence on lengthy manual clinical observations?

Main Objective

To develop an AI-based early ADHD screening system that analyzes speech, handwriting, visual attention, and cognitive behavior to help doctors detect potential ADHD indicators in children more efficiently.

Research Objectives

Specific Goals

🎯

Obj 1 · Eye Tracking

Develop a computer vision model to detect visual attention deficits through gaze patterns and fixation analysis during gamified tasks.

🎯

Obj 2 · Posture Detection

Build a pose estimation system to quantify hyperactivity and impulsivity through body movement patterns captured via webcam.

🎯

Obj 3 · Speech Analysis

Create an NLP/audio ML model to identify speech-language patterns including interruptions, pace anomalies, and word repetition associated with ADHD.

🎯

Obj 4 · Handwriting

Design a digital handwriting analysis module to assess fine motor control, pressure variation, and drawing consistency.

Technologies Used

Tech Stack

🐍

Python / FastAPI

Backend API server handling data transmission between the frontend interface and AI model layer using REST APIs and JSON format.

🤖

TensorFlow / PyTorch

Deep learning frameworks used for training and deploying the eye-tracking, posture, speech, and handwriting classification models.

👁️

OpenCV / MediaPipe

Computer vision libraries for real-time eye tracking, gaze estimation, and pose detection from webcam streams.

🎵

Librosa / Whisper

Audio processing and speech recognition libraries for extracting speech pattern features and transcribing child responses.

🗄️

MongoDB / Firebase

NoSQL database for secure storage of behavioral data, model predictions, and patient profiles.

⚛️

React.js

Child-friendly frontend interface built with React, providing an engaging, gamified experience for task completion.
